## Run with Javelit

[Javelit] is a Java framework inspired by Streamlit. After installed [Javelit] 
You can easily start the **Agent Handoff** sample app using the following command:

```
# jbang (recommended: install use jbang)
javelit src/test/java/JtMultiAgentHandoffApp 
```

```
# vanilla
java -jar javelit.jar src/test/java/JtMultiAgentHandoffApp 
```
